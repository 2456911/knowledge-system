三观：底层信念系统

世界观人生观价值观
三观是"底层操作系统" — 所有思考、判断、行动的总源头。回答：世界是什么样的？我该怎么活？什么选择更值得？

世界观

  • 底层逻辑是价值交换 — 任何关系、组织、市场都是
  • 世界是演化系统 — 永远在变化，适者生存
  • 规律不以人的意志为转移 — 尊重事实
应用：先问"这背后的价值交换是什么？""演化方向是什么？"

人生观

类比含义
行为日常经营
情绪短期股价波动
品格/底层模式商业模式 + 护城河
未来长期价值
每天问自己："今天的投入是在积累资产，还是在消耗资产？"

价值观

  • 底线绝对不动：不违法、不害人、不违心、不损人利己
  • 方法极致灵活：底线之内，穷尽一切办法找最优解
  • 算长期全局大账
  • 道德不是束缚，而是长期收益最高、风险最低的策略

三观与投资同构

三观投资类比核心问题
世界观对市场的底层认知世界有没有不变的底层规则？
人生观公司的使命与长期战略我的人生终极目标是什么？
价值观核心经营准则与风控红线什么钱不能赚？什么底线不能破？

元认知训练

元认知思考框架
元认知 = 观察自己的思考 + 校准自己的思考 + 不让情绪/本能替你做决定

四个可训练模块

一、第一性原理思考

事实层：真正发生了什么？有数据/证据吗？
逻辑层：推导过程对不对？有没有跳跃？
价值层：对我的核心资产有利还是有害？

二、人生CEO视角

资产负债
健康、能力、信用、投资、正向关系熬夜、情绪内耗、无效社交、高风险行为

三、内在锚定

在意别人看法时问：①这评价能决定我的长期价值吗？②我在做正确的事还是让他满意的事？③5年后回头看呢？

训练结果：别人夸你不飘，别人否定你不崩

四、思考前置校验

  1. 我在思考什么？
  2. 用什么方式思考？（归纳？演绎？情绪？）
  3. 证据/前提可靠吗？
  4. 有没有偏见/盲区？

高明的实用主义

道德长期主义
底线不动，方法灵活。道德是长期收益最高、风险最低的处事策略。

四条不可妥协的红线

不违法 · 不害人 · 不违心 · 不损人利己

四条落地规则

  1. 先划底线，再想办法 — 排除突破底线的方案，再穷尽正当方案
  2. 算大账不算小账 — 3年、5年后会带来什么？会不会埋雷？
  3. 优先找共赢方案 — 利他是长期利己的最可靠路径
  4. 守得住底线，放得开手脚

决策流程

1. 突破红线了吗？ → 突破了 → 不做
2. 长期全局最优方案是什么？
3. 有没有共赢方案？ → 有→优先 → 没有→选伤害最小的
4. 做完之后我睡得着吗？ → 睡不着→重新考虑

与自己周旋

自我接纳宁做我
"我与我周旋久，宁做我" — 做我，从来不是做那个剔除了所有弱点的完美自我，而是做那个完整的、真实的、哪怕带着不堪与阴影，依然坦然活着的本我。

5步周旋方法

01
停止敌我对抗
❌"我怎么这么嫉妒" → ✅"我此刻产生了嫉妒的情绪"。用第三人称称呼自己。
02
听懂情绪潜台词
①最强烈的情绪？②被什么触发？③在告诉我什么核心需求？
03
和平共处公约
接纳情绪，约束行为。嫉妒允许产生但不诋毁别人。
04
从纠错转向建设
停止缺点日记，开启闪光记录。种满鲜花，杂草自然没有空间。
05
周旋是常态
一辈子的事，没有一劳永逸的终点。今天和解明天可能又拉扯，这太正常了。

"宁做我"的三层含义

  1. 接纳完整的我 — 能慷慨付出的是你，会怕被消耗的也是你
  2. 不改掉所有缺点也配做自己 — 周旋的目的是和平共处，不是打赢
  3. 忠于内心，不迎合外界

心理学思维模型

心理学芒格
核心问题：人为什么会这么想、这么做？包括我自己。这是7大学科栅格里最常调用的学科。

核心认知偏差清单

偏差本质如何防止
确认偏误只看支持自己的证据主动寻找反面证据
损失厌恶亏100块的痛苦 > 赚100块的快乐关注期望值
锚定效应第一个数字锚定后续判断先独立判断
从众效应别人怎么做我也怎么做问"为什么这么做"
达克效应能力越低越自信承认无知

爱的艺术

不成熟的爱成熟的爱
"我爱你，因为我需要你"（索取型）"我需要你，因为我爱你"（给予型）
享乐适应：任何外部事件带来的幸福感都会在半年内回归基线 → 追求"幸福能力"而非"幸福事件"。

数学与概率

复利期望值安全边际
核心问题：这件事发生的真实概率是多少？长期期望值是正的还是负的？

核心模型

复利

  • 资金复利：长期稳定高ROE > 短期暴利
  • 能力复利：每天进步1%，1年后是37倍
  • 健康复利：每天运动30分钟，复利30年是质的飞跃

期望值 = 概率 × 回报

不要问"会成功吗"，而是问："成功概率×成功收益 - 失败概率×失败损失 = 值得做吗？"

安全边际

  • 好公司 + 低估值才是好投资
  • 不要满仓，不要借钱，保留弹药

均值回归

  • 任何极端状态都会回归平均值
  • 市场极度悲观时往往是买入时机

微观经济学与商业分析

护城河商业模式
核心问题：这门生意怎么赚钱？护城河在哪？值不值得长期持有？

4种真正的护城河

01
品牌心智
想到品类就想到你。例：可口可乐、茅台。
02
转换成本
换掉的代价极高。例：苹果生态。
03
网络效应
用的人越多越好用。例：微信。
04
成本优势
规模效应或资源独占。例：比亚迪。

机会成本

选择你放弃的
刷2小时手机= 看半本书/运动/陪家人
持有A股票= 持有B的收益
和消耗你的人在一起= 和滋养你的人的时间
"告诉我怎么激励你，我就能告诉你你会怎么行动" — 芒格

生物学与进化论

进化论适应
核心问题：什么样的特征能存活下来？我是在适应还是抗拒变化？

核心模型

适者生存

  • 商业世界 = 物竞天择，能适应的公司存活
  • 投资：这家公司5年后还能存活吗？
  • 健康：人体被设计来运动，久坐是反进化的

生态位

  • 每个物种占据独特生态位，不直接竞争
  • 我的独特生态位是什么？有没有在和自己不擅长的人硬碰硬？

二阶效应

  • 一阶：今天直接结果。二阶：由结果引发的下一层结果
  • 今天熬夜爽了 → 明天精力差→决策失误→长期损失

会计学与财务分析

三大报表ROE
核心问题：赚的是真钱还是假钱？资产质量如何？同样适用于分析人。

三大报表

报表回答核心指标
利润表是否赚钱？营收、净利润、毛利率
资产负债表是否健康？负债率、流动比率
现金流量表利润是真是假？经营性现金流、自由现金流
铁律：看利润不看现金流 = 被财务造假骗。

ROE

ROE = 净利润 ÷ 净资产 → >15%为优秀

识人财报类比

财务概念识人类比
利润表跟你在一起是充电还是耗电？
资产负债表情绪负债多吗？
现金流量表说的和做的一致吗？
ROE相处的时间ROE是正的吗？

哲学与逻辑学

第一性原理逆向思维
核心问题：这个结论的前提可靠吗？有没有更根本的解法？

第一性原理

拆解一切表象，回到不可再拆分的基本事实，从这里重新推导。
  1. 抛开所有既有结论和别人看法
  2. 只看最基本事实：本质是什么？
  3. 从基本事实重新推导

反面思考

"反过来想，一定要反过来想" — 芒格
  • 不只问"怎么成功"，也问"怎么失败"
  • 投资：如果我会亏，最可能因为什么？
  • 识人：这个人翻脸时最差会怎样？

奥卡姆剃刀

如无必要，勿增实体。解释一件事，优先选最简单的。

历史学与社会学

周期人性不变
核心问题：历史上类似的情况怎么发展的？人性不变，我是否在重蹈覆辙？

周期理论

  • 市场有周期，经济有周期，人生也有周期
  • 当大家说"这次不一样"时，往往是周期顶部
  • 低谷不是终点，是下一个高峰的起点

人性不变

  • 技术在变，但人性从未变：贪婪、恐惧、从众、短视
  • 分析市场情绪时，不看新闻，看人性

幸存者偏差

  • 看到的成功案例是从海量失败中筛选出来的
  • 某个投资方法看起来很成功，但幸存者偏差了多少失败者？

健康决策原则

健康芒格纳瓦尔
核心规则：健康是第一核心资产，是一切复利的载体。没有健康，财富和幸福都归零。
芒格："我认识的聪明人没有一个不每天锻炼的。没有例外。"

四大健康支柱

💪
运动（每天30分钟）
芒格铁律：每天运动没有例外。心率60%-70%。工作时段10:30/14:30/16:00各10分钟跳绳。
😴
睡眠（7.5小时）
23:00入睡|7:00起床，5个完整90分钟周期。睡前1小时关电子设备。
🥗
饮食（7分饱+低糖低油）
早：高蛋白+慢碳。午：7分饱。晚：清淡少碳水。上午1杯黑咖啡，下午2点后不碰。
⚡
精力管理
最高精力→自我提升。中等→工作保达标。剩余→休息。禁忌：把最高精力给工作。

失控恢复流程

1. 暂停觉察（3次深呼吸）→ 我在逃避什么？
2. 回到底层逻辑 → 健康是一切的基础
3. 小步行动 → 今天走15分钟/今晚不熬夜
4. 一次失控 ≠ 前功尽弃 → 尽快回来

识人决策原则

识人价值投资
核心规则：人 = 终身上市公司。看人只看长期价值（商业模式），不看短期情绪（股价）。

识人5步法

01
剥离表象
不听他说什么，只看他做什么。嘴炮=财务造假。
02
锚定驱动力
求安全/求利益/求意义自尊。利益冲突场景最能看清。
03
验证底盘
人品+情绪稳定性+言行一致。无底线=无护城河。
04
回看历史
3-5年一贯表现 > 当下态度 > 一次高光。
05
划定仓位
90-100:核心重仓 | 70-89:持有 | 60-69:轻仓 | <60:清仓

维度5分3分0分

情绪现金流(25)鼓励、松弛感中性负能量爆棚
护城河人品(20)原则性极强大是非不含糊不择手段
历史业绩(15)一贯优秀有好有坏一贯拉胯

关系与幸福原则

关系幸福爱
核心规则：先修己再爱人，以给予为核心，以尊重为前提。幸福是活成了对的人之后自然而然的馈赠。

爱的三条铁律

01
先修己再爱人
你给不出你没有的东西。自己没有安全感→只能通过控制获得。

幸福的三个维度

维度核心
对内：内核稳定锚点从"外求"转向"内生"。没有内耗=最大幸福。
对外：关系滋养精准识人避雷，筛选滋养型关系。
对事：持续成长人生价值随时间复利增长。活在当下。

面对痛苦的三步法

  1. 先接纳，不抗拒 — 痛苦是常态
  2. 再拆解，不焦虑 — 可控的穷尽办法，不可控的彻底接纳
  3. 终迭代，不浪费 — 把痛苦变成优化操作系统的机会
享乐适应：外部事件带来的幸福感半年内回归基线 → 这套体系给你的不是一次性快乐，而是持续创造幸福的能力。

自我与他者

  1. 坦然接纳"渴望被看见"的本能
  2. 自我感受第一，他人认可第二
  3. 留一个"无人看见也愿意做的事"作为压舱石
地狱不是他人本身，而是你自愿戴上他人凝视的镣铐。

自我管理原则

精力管理作息
核心规则：把每日最高质量的纯净精力优先留给自我提升，工作仅保达标不内卷。

精力分配铁律

最高质量精力 → 自我提升（学习、运动、思考）
       ↓
  中等精力 → 工作（保达标，不追求完美）
       ↓
  剩余精力 → 休息放松

⚠️ 禁忌：把最高精力给工作，然后"太累了"不学习不运动

认知体系不是一次性构建的，而是通过持续复盘迭代不断进化。

四级迭代节奏

节奏时机核心动作
日复盘每晚22:30（10分钟）觉察被带跑、挖动机、下次怎么拦
周检视每周日晚（30分钟）精力ROE、资产负债表更新、原则校准
季度更新每季末（2小时）体系升级、淘汰无效原则、新增新原则
年度总结每年末（半天）方向校准、核心命题复核

好的复盘 vs 坏的复盘

坏的好的
罗列事件挖到底层动机和模型失效原因
"要更努力""下次遇到X情况，我按Y原则行动"
只复盘失败也复盘成功（哪个模型奏效了？）
复盘但不更新体系每次都有具体的体系更新动作

体系进化信号

  • 遇到新情况，现有原则无法覆盖 → 添加新原则
  • 某条原则反复无效 → 淘汰或修改
  • 对某个概念有更深的理解 → 更新笔记
  • 同样错误超过3次 → 原则没内化，换方式训练
